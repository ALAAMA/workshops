CPS DWS Data

CEPR Uniform Extracts of the CPS Displaced Worker Survey are available in Stata 8/9 format, for every other year during 1994-2010.

The latest version of the CEPR CPS DWS Uniform Extracts is Version 1.02:

  • cepr_dws_2010_dta.zip
  • cepr_dws_2008_dta.zip
  • cepr_dws_2006_dta.zip
  • cepr_dws_2004_dta.zip
  • cepr_dws_2002_dta.zip
  • cepr_dws_2000_dta.zip
  • cepr_dws_1998_dta.zip
  • cepr_dws_1996_dta.zip
  • cepr_dws_1994_dta.zip

Each compressed file is about 5Mb; uncompressed, each Stata data file is about 35Mb. If you don’t want to download the data, you can replicate the CEPR CPS Job Tenure Uniform Extracts by using our programs. For data in other formats, please contact us.

ATTRIBUTION

If use the CEPR Uniform Extracts of the CPS Displaced Worker Survey in your research, please cite them as

  • Center for Economic and Policy Research. 2012. CPS Displaced Worker Uniform Extracts, Version 1.02. Washington, DC.

The timestamp and version of your downloaded dataset can be found using the Stata –describe, short–command.

Please contact us if you publish any research using this data, so we can list your work on our CEPR Uniform Extracts publications page.
